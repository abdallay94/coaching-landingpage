# Coaching Landing Page

This repository is structured for GitHub Pages hosting.

## Files

- `index.html` is the website homepage. GitHub Pages loads this file automatically.

## Publish On GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
5. Choose the `main` branch and the `/ (root)` folder.
6. Click **Save**.

After GitHub finishes publishing, your site will be available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/coaching-landingpage/
```

If you rename the repository, replace `coaching-landingpage` in the URL with the new repository name.
